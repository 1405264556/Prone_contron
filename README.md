# 无人机 Qt6 可视化上位机

基于 PySide6 / Qt6 的 WiFi UFO 协议无人机地面站程序，集图传接收、飞行控制、飞控串口调试于一体。提供开箱即用的 Windows 可执行文件，无需安装 Python 环境。

> **安全警告**：测试飞控、起飞、降落、方向控制前，请务必先拆桨或固定机体。程序默认锁定所有飞行动作，必须手动勾选安全确认后才会发出控制指令。

### 四、右侧标签页

#### 📊 监控标签页

- **快速状态卡片**（3 个横向排列）：
  - WiFi 状态：等待连接（红）/ 已连接（绿）
  - 飞控状态：未连接（红）/ 已连接（绿）
  - 控制锁定状态：已锁定（橙）/ 已解锁（绿）
- **遥测数据表**：字段/值两列，动态更新。包含无人机 IP、WiFi 协议、串口飞控型号、安全锁状态、视频统计、飞控遥测（固件版本、电压、CPU、陀螺仪等）

#### 🎮 飞行控制标签页

- **安全锁区域**：
  - 复选框："我已拆桨/固定机体，并确认允许发送飞行动作"
  - 未勾选时所有飞行指令被阻止，弹出警告对话框
- **当前命令状态**：实时显示最后发送的控制指令
- **遥控器式快捷控制**（3×3 按钮网格）：
  ```
  ⇧升高(R)    ↑前进(W)    ↻右旋(E)
  ←左移(A)    ◎悬停(Space) →右移(D)
  ⇩降低(F)    ↓后退(S)    ↺左旋(Q)
  ```
  每个按钮发送约 0.45 秒的短促脉冲，松开后自动回中。
- **连续控制量滑块**（4 个水平滑块）：
  - 横滚 Roll / 俯仰 Pitch / 油门 Throttle / 偏航 Yaw
  - 范围 0-255，中位 128
  - 拖动滑块实时更新数值标签，并推送到后台控制线程
- **动作按钮**：
  - ▶ 启动连续控制：以 50ms 间隔持续发送滑块位置
  - ⏹ 停止连续控制：停止发送
  - ◎ 悬停/回中：全部滑块复位到 128
  - ⬇ 油门归零：油门设为 0
  - 🛫 起飞 / 🛬 降落：连续发送 1 秒指令脉冲（需二次确认）
  - ⚠ 急停：红色危险按钮，发送急停指令（需二次确认）

## 使用流程

### 基本操作流程

1. **安全准备**：拆掉螺旋桨，或用夹具固定无人机
2. **连接 WiFi**：将电脑连接到无人机热点 `WiFiUFO-3BE7F2`
3. **启动程序**：双击 `DroneGroundStation.exe`
4. **检查配置**：
   - 协议选择 `WiFi UFO UDP`
   - IP 确认 `192.168.0.1`
   - 远端/本地端口均为 `40000`
5. **启动图传**：点击「UFO 心跳」按钮，等待视频预览画面出现
6. **串口调试**（可选）：
   - 选择串口（如 `COM9`），波特率 `115200`
   - 点击「连接串口」
   - 切换到「开发者界面」发送 CLI 命令
7. **飞行控制**：
   - 勾选安全锁复选框
   - 使用方向按钮或键盘快捷键进行短促控制
   - 或拖动滑块后点击「启动连续控制」进行精细操控
   - 起飞/降落/急停操作需二次确认

### 典型工作场景

| 场景 | 操作 |
|------|------|
| 仅查看图传 | 连接 WiFi → 打开程序 → 点 UFO 心跳 |
| 调试飞控参数 | 连接串口 → 切换开发者界面 → 发送 `dump` 命令 |
| 手动飞行 | 解安全锁 → 拖动四个滑块 → 启动连续控制 → 实时调节 |
| 紧急情况 | 直接点击红色「⚠ 急停」按钮 → 确认 → 电机立即停止 |

## 控制协议说明

WiFi UFO 协议基于对 `WiFiUFO-3BE7F2` 无人机的抓包分析得出。

### 心跳包

```
63 63 01 00 00 00 00
│  │  │  └────────── 保留字段
│  │  └── 包类型 = 0x01（信息/心跳）
│  └── 协议标识
└── 协议标识
```

### 控制包

```
63 63 0a 00 00 08 00 66 [R] [P] [T] [Y] [M] [X]
│  │  │  └─┬─┘ └─┬─┘ └┬┘  │   │   │   │   │  │
│  │  │    │     │     │   │   │   │   │   │  └── XOR 校验（字节 8-12）
│  │  │    │     │     │   │   │   │   │   └── 模式（0=普通 1=起飞 2=降落 4=急停）
│  │  │    │     │     │   │   │   │   └── Yaw 偏航（0-255，128 中位）
│  │  │    │     │     │   │   │   └── Throttle 油门（0-255，128 中位）
│  │  │    │     │     │   │   └── Pitch 俯仰（0-255，128 中位）
│  │  │    │     │     │   └── Roll 横滚（0-255，128 中位）
│  │  │    │     │     └── 固定值 0x66
│  │  │    │     └── 保留
│  │  │    └── 消息长度
│  │  └── 包类型 = 0x0a（控制包）
│  └── 协议标识
└── 协议标识
```

### 控制值说明

| 通道 | 范围 | 中位 | 说明 |
|------|------|------|------|
| Roll | 0-255 | 128 | <128 左倾，>128 右倾 |
| Pitch | 0-255 | 128 | <128 后仰，>128 前倾 |
| Throttle | 0-255 | 128 | 0=停转，128=悬停油门，255=满油门 |
| Yaw | 0-255 | 128 | <128 左旋，>128 右旋 |

### 数据包类型汇总

| 类型字节 | 含义 |
|----------|------|
| `01` | WiFi UFO 信息包（含 SSID 等设备信息） |
| `03` | 视频数据分片包 |
| `0a` | 控制指令包 |

## 键盘快捷键

无需鼠标，所有飞行操作均可通过键盘完成：

| 按键 | 功能 | 对应按钮 |
|------|------|----------|
| `W` / `↑` | 前进（Pitch 176） | ↑ 前进 |
| `S` / `↓` | 后退（Pitch 80） | ↓ 后退 |
| `A` | 左移（Roll 80） | ← 左移 |
| `D` | 右移（Roll 176） | → 右移 |
| `Q` | 左旋（Yaw 80） | ↺ 左旋 |
| `E` | 右旋（Yaw 176） | ↻ 右旋 |
| `R` | 升高（Throttle 176） | ⇧ 升高 |
| `F` | 降低（Throttle 80） | ⇩ 降低 |
| `Space` | 悬停/回中 | ◎ 悬停 |

> 按键约 0.45 秒后自动回中（悬停为 0.25 秒），方向按钮行为一致。


```powershell

.\.venv_qt6\Scripts\pip install pyinstaller

.\.venv_qt6\Scripts\pyinstaller.exe --onefile --windowed --name DroneGroundStation --distpath .\dist --workpath .\build .\qt_ground_station\python_qt6\qt_drone_station.py
```

## 技术架构

```
┌─────────────────────────────────────────────────────┐
│                    Qt6 主界面 (MainWindow)           │
│  ┌──────────┐  ┌──────────┐  ┌──────────────────┐  │
│  │ 连接配置  │  │ 图传预览  │  │ 监控/控制/CLI/说明│  │
│  │ Udp+串口 │  │ + 日志   │  │    (QTabWidget)  │  │
│  └──────────┘  └──────────┘  └──────────────────┘  │
└───────────────┬──────────────┬──────────────────────┘
                │              │
        ┌───────▼──────┐ ┌────▼──────────────┐
        │ WifiUfoWorker │ │ CleanflightSerial │
        │   (QThread)   │ │ Client (QThread)  │
        │               │ │                   │
        │ • UDP I/O     │ │ • 串口 I/O        │
        │ • 视频重组    │ │ • CLI 命令收发    │
        │ • 控制发送    │ │ • 遥测解析        │
        │ • 50ms 定时器 │ │                   │
        └───────┬──────┘ └────┬──────────────┘
                │              │
        ┌───────▼──────┐ ┌────▼──────────────┐
        │ WiFi UFO UDP │ │ Cleanflight 串口  │
        │ 192.168.0.1  │ │ COM9 @ 115200    │
        │   :40000     │ │                   │
        └──────────────┘ └───────────────────┘
```

**线程模型**：
- **主线程**：Qt 事件循环，处理所有 UI 更新
- **WifiUfoWorker 线程**：UDP 收发 + 视频分片重组 + 控制包定时发送
- **CleanflightSerialClient 线程**：串口读写 + CLI 交互
- 线程间通过 Qt **信号/槽**（Signal/Slot）机制通信，线程安全

**视频处理流程**：
1. UDP 接收 802 字节数据包（含 54 字节头部 + JPEG 分片）
2. 按分片序号 `fragment_index` 排序重组
3. 遇到 `FF D8`（SOI）开始新帧，`FF D9`（EOI）结束帧
4. 校验帧尺寸一致性，丢弃尺寸突变的坏帧
5. 约 18 FPS 限帧渲染到 QLabel，减少彩色闪烁伪影
